# Hasta Takip Sistemi (HTS) 🏥

C# Windows Forms ve SQL Server kullanılarak geliştirilmiş, hastane veya klinik ortamlarında hasta kayıtlarını ve durumlarını dinamik olarak yönetmeyi sağlayan masaüstü tabanlı bir **Hasta Takip Sistemi** otomasyonudur.

## 🚀 Özellikler

* **Güvenli Giriş Sistemi:** Kullanıcı adı ve şifre doğrulamalı yetkilendirme.
* **Tam CRUD İşlemleri:** Hasta kayıtlarını ekleme, listeleme, güncelleme ve silme.
* **Gelişmiş SQL Sorguları ve İstatistik Paneli:** Veri tabanından anlık çekilen verilerle dinamik raporlama:
    * Toplam kayıtlı hasta sayısı (`COUNT`)
    * Hastaların yaş ortalaması (`AVG`)
    * Cinsiyete göre hasta dağılımı (Erkek / Kadın sayıları)
    * Kritik durumdaki veya eks (`ex`) olan hasta istatistikleri.
* **Kullanıcı Dostu Arayüz:** `TableLayoutPanel` ile organize edilmiş, responsive form tasarımları.

## 🛠️ Kullanılan Teknolojiler

* **Geliştirme Ortamı:** Visual Studio 2022
* **Programlama Dili:** C# (.NET Framework)
* **Arayüz Tasarımı:** Windows Forms (WinForms)
* **Veri Tabanı:** SQL Server 2019 / 2022
* **Veri Erişimi:** ADO.NET (`SqlConnection`, `SqlCommand`, `SqlDataReader`)
